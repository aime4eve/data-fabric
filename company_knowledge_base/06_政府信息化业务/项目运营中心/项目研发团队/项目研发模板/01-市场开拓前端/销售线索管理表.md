## 线索质量评估

### 线索评分标准
```
评分维度 (总分100分)：
• 需求明确度 (25分)
  - 需求清晰：20-25分
  - 需求模糊：10-19分
  - 无明确需求：0-9分

• 预算匹配度 (20分)
  - 预算充足：16-20分
  - 预算一般：10-15分
  - 预算不足：0-9分

• 决策权限 (20分)
  - 决策者直接接触：16-20分
  - 影响者接触：10-15分
  - 执行者接触：0-9分

• 时间紧迫性 (15分)
  - 急需解决：12-15分
  - 计划中：8-11分
  - 无时间要求：0-7分

• 竞争优势 (10分)
  - 明显优势：8-10分
  - 一般优势：5-7分
  - 无明显优势：0-4分

• 关系基础 (10分)
  - 关系良好：8-10分
  - 关系一般：5-7分
  - 无关系基础：0-4分
```

### 线索质量分级
```
A级线索 (80分以上)：
• 高优先级跟进
• 分配给资深销售
• 每周至少跟进2次

B级线索 (60-79分)：
• 中等优先级跟进
• 分配给经验销售
• 每周至少跟进1次

C级线索 (40-59分)：
• 一般优先级跟进
• 可分配给初级销售
• 每两周跟进1次

D级线索 (40分以下)：
• 低优先级或暂停跟进
• 定期回访确认状态变化
• 每月跟进1次
```
