# Quality Checklist

- [ ] **Responsiveness**
    - [ ] Mobile (320px+): No horizontal scroll, readable text.
    - [ ] Tablet (768px+): Grid layouts adjust correctly (2 cols).
    - [ ] Desktop (1024px+): Full layout (3-4 cols), proper spacing.

- [ ] **UI/UX**
    - [ ] Hover states on all buttons and cards.
    - [ ] Navigation is sticky and accessible.
    - [ ] Text contrast meets accessibility standards.
    - [ ] Swiper carousel slides smoothly.

- [ ] **Code Quality**
    - [ ] Semantic HTML tags used.
    - [ ] Code comments added (Author: 伍志勇).
    - [ ] No inline CSS (use Tailwind classes).
    - [ ] Source citations added if external content used.
